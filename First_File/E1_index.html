Suunnan ottaminen
  • Ensin tulee määritellä oma pisteensä
  • Sitten tulee määritellä missä päin pohjoinen on kartalla
  • Tämän jälkeen otetaan suunta kartalla haluuttuun pisteeseen:
    •
    1. Asetetaan kompassin kulkusuuntanuoli lähtöpisteestä määränpäähän
    2. Käännetään kompassin neularasiaa niin, että kompassin pohjoishaarukka osoittaa kohti pohjoista
    3. Käännytään oikeaan suuntaan kunnes kompassin punainen neula on pohjoishaarukan sisällä
    4. Kuljetaan kohti haluttua suuntaa lukien kartaa, maastoa sekä kompassia
